Super Smart Q&A Bot
-----------
전체적인 로직
---------
- 상대가 질문을 하면 그에 대한 검색내용을 핵심키워드, 대답과 함께 학습하고, 상대가 원하는 내용을 알려준다.

- 질문에 대한 답변을 이미 알고 있으면 학습한 내용중에 질문자가 가장 알고 싶어하는 핵심 요소를 파악하며 답변한다.

- 정보 파일은 json 파일로 저장함.

- Shout Find  
이미 알고 있는 지식들을 클래스로 하나하나 구분하고,  
지식을 찾는 역할을 하는 클래스가 지식들의 집합 클래스에 인풋을 넣으먼 집합들이 자동적으로 반응하는 기술을 만든다.

개발 순서
---------
|자연어처리|설명|개발상태|
|:---:|:---:|:---:|
|키워드 추출|gensim 라이브러리의 keywords 모듈 사용|완료|
|글 요약|본문과 본문 속 문장을 키워드를 기준으로 벡터라이저 후에 벡터라이저된 데이터로 본문과 문장의 코사인 유사도가 일정값 이하이면 본문 속에서 해당 문장을 제거.|완료|

Shout Find  (  기획중.  )

- Shouter 역할 개발해야함

- Listener 역할 개발해야함

- 알고리즘 문제에 먼저 사용해봐야 함.

- 게임 캐릭터는 여러명이여도 캐릭터마다 동시에 연산처리가 가능하기 때문에, 그 원리를 찾아봐야한다. (시물레이션 원리)

- 먼저 ShoutFind 를 유니티를 이용하여 시뮬레이션으로 간단하게 구현해보는게 좋을 것 같음.
