Super Smart Q&A Bot
-----------
전체적인 로직
---------
- 상대가 질문을 하면 그에 대한 검색내용을 핵심키워드, 대답과 함께 학습하고, 상대가 원하는 내용을 알려준다.

- 질문에 대한 답변을 이미 알고 있으면 학습한 내용중에 질문자가 가장 알고 싶어하는 핵심 요소를 파악하며 답변한다.

- 정보 파일은 json 파일로 저장함.

- Shout Find  
이미 알고 있는 지식들을 클래스로 하나하나 구분하고,  
지식을 찾는 역할을 하는 클래스가 지식들의 집합 클래스에 인풋을 넣으먼 집합들이 자동적으로 반응하는 기술을 만든다.

- 학습할 때에는 객관적인 글, 주관적인 글을 구분하여 객관적인 글만 학습.
-> 주관적인 글에는 잘못된 정보가 많을 수 있기 때문.


개발 순서
---------
1. 자연어 처리  (  2/3  )
- 키워드 추출 : gensim 라이브러리의 keywords 모듈 사용  (  완료. )  
- 글 요약 (  완료.  )
    - 본문, 본문 속 문장을 키워드를 기준으로 벡터라이저 후에 벡터라이저된 데이터로 본문과 문장의
    코사인 유사도가 일정값 이하이면 본문 속에서 해당 문장을 제거.
    - 순서대로 정리.  
- 글 종류 분류  (  진행중.  )
    - 감정분석  (  완료.  )
        - 너무 긍정적이거나, 너무 부정적일때. 주관적일 가능성 ⬆
    - 글 종류 분류 관련 자료 : https://arxiv.org/ftp/arxiv/papers/1312/1312.6962.pdf, https://www.cs.utah.edu/~riloff/pdfs/cicling05.pdf
2. Shout Find  (  기획중.  )
- LiveData 원리 익히기
