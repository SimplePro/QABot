Super Smart Q&A Bot
-----------

- 상대가 질문을 하면 그에 대한 검색내용을 핵심키워드, 대답과 함께 학습하고, 상대가 원하는 내용을 알려준다.

- 질문에 대한 답변을 이미 알고 있으면 학습한 내용중에 질문자가 가장 알고 싶어하는 핵심 요소를 파악하며 답변한다.

- 정보 파일은 json 파일로 저장함.

- Shout Find  
이미 알고 있는 지식들을 클래스로 하나하나 구분하고,  
지식을 찾는 역할을 하는 클래스가 지식들의 집합 클래스에 인풋을 넣으먼 집합들이 자동적으로 반응하는 기술을 만든다.

- 글을 정리할때 주장하는 글, 설명하는 글 등으로 나누고 정리해야함. (글의 종류를 구분하는 인공지능을 개발해야 함.)
-> 잘못된 정보나 주관적인 내용은 걸러야 하기 때문.
