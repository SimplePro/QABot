Q&A Bot
-----------
전체적인 로직
---------
- 상대가 질문을 하면 그에 대한 검색내용을 핵심키워드, 대답과 함께 학습하고, 상대가 원하는 내용을 알려준다.

- 질문에 대한 답변을 이미 알고 있으면 학습한 내용중에 질문자가 가장 알고 싶어하는 핵심 요소를 파악하며 답변한다.

- 정보 파일은 json 파일로 저장함.

- 위키백과에서 summary 한 후에 문장들마다 단어와 함께 학습하고, 질문자가 추후에 뭔가 구체적으로 질문을 하면, 질문에 있는 단어와 가장 높은 유사도의 데이터로 대답한다.

- 위키백과에서 summary 한 후에 문장들마다 단어의 집합으로 바꾸고, 질문에 있는 단어와 가장 높은 유사도의 문장을 반환한다.

- 여기서 단어들은 Words Graph 를 이용하여 그래프 데이터로 학습하면 좋을 것 같음.

---------
|자연어처리|설명|개발상태|
|:---:|:---:|:---:|
|키워드 추출|gensim 라이브러리의 keywords 모듈 사용|완료|
|글 요약|본문과 본문 속 문장을 키워드를 기준으로 벡터라이저 후에 벡터라이저된 데이터로 본문과 문장의 코사인 유사도가 일정값 이하이면 본문 속에서 해당 문장을 제거.|완료|
